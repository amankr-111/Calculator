                            Calculator
This is a basic calculator web application built using HTML, CSS, and JavaScript.

                                Usage
To use the calculator, simply click on the buttons to input numbers and operators. The calculator supports basic arithmetic operations such as addition, subtraction, multiplication, and division. To clear the screen, click on the "C" button.
